=============================

.. rst-class:: classref-introduction-group

描述
----

模拟阈值绑定修改器能够根据指定的阈值将 float 输入修改为布尔值输入。

详见 `XR_VALVE_analog_threshold <https://registry.khronos.org/OpenXR/specs/1.1/html/xrspec.html#XR_VALVE_analog_threshold>`__\ 。
