ORMMaterial3D
=============

**Успадковує:** :ref:`BaseMaterial3D<class_BaseMaterial3D>` **<** :ref:`Material<class_Material>` **<** :ref:`Resource<class_Resource>` **<** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

PBR (Фізично на основі рендерингу) матеріал для використання на предметах 3D. Використання текстури ORM.

.. rst-class:: classref-introduction-group

Опис
--------

ORMMaterial3D властивості успадковуються з :ref:`BaseMaterial3D<class_BaseMaterial3D>`. На відміну від :ref:`StandardMaterial3D<class_StandardMaterial3D>`, ORMMaterial3 D використовує єдину текстуру для прикусу навколишнього середовища, грубості та металевих карт, відомих як текстура ORM.
