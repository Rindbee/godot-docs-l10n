HSeparator
==========

**Успадковує:** :ref:`Separator<class_Separator>` **<** :ref:`Control<class_Control>` **<** :ref:`CanvasItem<class_CanvasItem>` **<** :ref:`Node<class_Node>` **<** :ref:`Object<class_Object>`

Горизонтальна лінія, яка використовується для відокремлення інших елементів керування.

.. rst-class:: classref-introduction-group

Опис
--------

Горизонтальний роздільник, який використовується для відокремлення інших елементів керування, розташованих **вертикально**. **HSeparator** є суто візуальним і зазвичай малюється як :ref:`StyleBoxLine<class_StyleBoxLine>`.
