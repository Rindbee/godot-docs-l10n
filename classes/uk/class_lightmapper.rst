Lightmapper
===========

**Успадковує:** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

Опис
--------

Цей клас повинен бути розширений за допомогою спеціальних Lightmapper класів. Lightmappers можна використовувати за допомогою :ref:`LightmapGI<class_LightmapGI>` для швидкого запікання глобального освітлення в 3D.

Godot містить вбудований світломагнітач на основі GPU :ref:`LightmapperRD<class_LightmapperRD>`, який використовує обчислювальні шейкери, але користувацькі блискавки можуть бути реалізовані за допомогою модулів C++.
