AnimationRootNode
=================

**Успадковує:** :ref:`AnimationNode<class_AnimationNode>` **<** :ref:`Resource<class_Resource>` **<** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

Базовий клас для :ref:`AnimationNode<class_AnimationNode>`, який містить одну або декілька складених анімацій. Зазвичай використовується для :ref:`AnimationTree.tree_root<class_AnimationTree_property_tree_root>`.

Опис
--------

**AnimationRootNode** — це базовий клас для :ref:`AnimationNode<class_AnimationNode>`, який містить повну анімацію. Повна анімація стосується виводу :ref:`AnimationNodeOutput<class_AnimationNodeOutput>` у :ref:`AnimationNodeBlendTree<class_AnimationNodeBlendTree>` або виводу іншого **AnimationRootNode**. Використовується для :ref:`AnimationTree.tree_root<class_AnimationTree_property_tree_root>` або в інших **AnimationRootNode**.

 Приклади вбудованих кореневих вузлів включають :ref:`AnimationNodeBlendTree<class_AnimationNodeBlendTree>` (дозволяє змішувати вузли між собою за допомогою різних режимів), :ref:`AnimationNodeStateMachine<class_AnimationNodeStateMachine>` (дозволяє налаштовувати змішування та переходи між вузлами за допомогою шаблону кінцевого автомата), :ref:`AnimationNodeBlendSpace2D<class_AnimationNodeBlendSpace2D>` (дозволяє лінійне змішування між **три** :ref:`AnimationNode<class_AnimationNode>`\ s), :ref:`AnimationNodeBlendSpace1D<class_AnimationNodeBlendSpace1D>` (дозволяє лінійне змішування лише між **двома** :ref:`AnimationNode<class_AnimationNode>`\ s).
