TextServerExtension
===================

**Успадковує:** :ref:`TextServer<class_TextServer>` **<** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

**Успадковано від:** :ref:`TextServerAdvanced<class_TextServerAdvanced>`, :ref:`TextServerDummy<class_TextServerDummy>`, :ref:`TextServerFallback<class_TextServerFallback>`

Базовий клас для користувацького :ref:`TextServer<class_TextServer>`.

.. rst-class:: classref-introduction-group

Опис
--------

Зовнішній :ref:`TextServer<class_TextServer>` виконання повинні успадкуватися від цього класу.
