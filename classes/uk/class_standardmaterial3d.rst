StandardMaterial3D
==================

PBR (Фізично на основі рендерингу) матеріал для використання на предметах 3D.

.. rst-class:: classref-introduction-group

Опис
--------

**StandardMaterial3D** властивості успадкуються від :ref:`BaseMaterial3D<class_BaseMaterial3D>`. **StandardMaterial3D** використовує окремі текстури для з'єднання, грубості та металевих карт. Для використання однієї карти ORM для всіх 3 текстур, скористайтеся :ref:`ORMMaterial3D<class_ORMMaterial3D>`.
