VideoStreamTheora
=================

**Успадковує:** :ref:`VideoStream<class_VideoStream>` **<** :ref:`Resource<class_Resource>` **<** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

:ref:`VideoStream<class_VideoStream>` ресурс для Ogg Theora відео.

.. rst-class:: classref-introduction-group

Опис
--------

Ресурс :ref:`VideoStream<class_VideoStream>`, що обробляє відеоформат `Ogg Theora <https://www.theora.org/>`__ з розширенням ``.ogv``. Кодек Theora декодується на процесорі.

\ **Примітка:** Хоча відео Ogg Theora також можуть мати розширення ``.ogg``, вам доведеться перейменувати розширення на ``.ogv``, щоб використовувати ці відео в Godot.
