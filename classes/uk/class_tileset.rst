TileSet
=======

**Успадковує:** :ref:`Resource<class_Resource>` **<** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

Бібліотека тайлів для карт тайлів.

.. rst-class:: classref-introduction-group

Опис
--------

TileSet – це бібліотека тайлів для :ref:`TileMapLayer<class_TileMapLayer>`. TileSet обробляє список :ref:`TileSetSource<class_TileSetSource>`, кожен з яких зберігає набір тайлів.

Тайли можуть бути або з :ref:`TileSetAtlasSource<class_TileSetAtlasSource>`, який рендерить тайли з текстури з підтримкою фізики, навігації тощо, або з :ref:`TileSetScenesCollectionSource<class_TileSetScenesCollectionSource>`, який надає тайли на основі сцени.

Посилання на тайли здійснюється за допомогою трьох ідентифікаторів: їх вихідний ідентифікатор, ідентифікатор координат атласу та їх альтернативний ідентифікатор тайла.

TileSet можна налаштувати таким чином, щоб його тайли надавали більше або менше властивостей. Для цього ресурси TileSet використовують шари властивостей, які ви можете додавати або видаляти залежно від ваших потреб.

Наприклад, додавання фізичного шару дозволяє надавати вашим тайлам форми зіткнень. Кожен шар має спеціальні властивості (фізичний шар і маску), тому ви можете додати кілька фізичних шарів TileSet для кожного потрібного вам типу зіткнень.

Дивіться функції для додавання нових шарів для отримання додаткової інформації.
