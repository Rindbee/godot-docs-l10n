–†–µ–≥—É–ª—è—Ä–Ω–∏–π –≤–∏—Ä–∞–∑ (–∞–±–æ regex) ‚Äì —Ü–µ –∫–æ–º–ø–∞–∫—Ç–Ω–∞ –º–æ–≤–∞, —è–∫—É –º–æ–∂–Ω–∞ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ –¥–ª—è —Ä–æ–∑–ø—ñ–∑–Ω–∞–≤–∞–Ω–Ω—è —Ä—è–¥–∫—ñ–≤, —â–æ –≤—ñ–¥–ø–æ–≤—ñ–¥–∞—é—Ç—å –ø–µ–≤–Ω–æ–º—É —à–∞–±–ª–æ–Ω—É, —Ç–∞–∫–æ–º—É —è–∫ URL-–∞–¥—Ä–µ—Å–∏, –∞–¥—Ä–µ—Å–∏ –µ–ª–µ–∫—Ç—Ä–æ–Ω–Ω–æ—ó –ø–æ—à—Ç–∏, –ø–æ–≤–Ω—ñ —Ä–µ—á–µ–Ω–Ω—è —Ç–æ—â–æ. –ù–∞–ø—Ä–∏–∫–ª–∞–¥, —Ä–µ–≥—É–ª—è—Ä–Ω–∏–π –≤–∏—Ä–∞–∑ ``ab[0-9]`` –∑–Ω–∞–π–¥–µ –±—É–¥—å-—è–∫–∏–π —Ä—è–¥–æ–∫, —è–∫–∏–π —Å–∫–ª–∞–¥–∞—î—Ç—å—Å—è –∑ ``ab``, –∑–∞ —è–∫–∏–º –π–¥–µ –±—É–¥—å-—è–∫–µ —á–∏—Å–ª–æ –≤—ñ–¥ ``0`` –¥–æ ``9``. –î–ª—è –±—ñ–ª—å—à –¥–µ—Ç–∞–ª—å–Ω–æ–≥–æ –æ–∑–Ω–∞–π–æ–º–ª–µ–Ω–Ω—è –≤–∏ –º–æ–∂–µ—Ç–µ –ª–µ–≥–∫–æ –∑–Ω–∞–π—Ç–∏ —Ä—ñ–∑–Ω—ñ –Ω–∞–≤—á–∞–ª—å–Ω—ñ –ø–æ—Å—ñ–±–Ω–∏–∫–∏ —Ç–∞ –¥–µ—Ç–∞–ª—å–Ω—ñ –ø–æ—è—Å–Ω–µ–Ω–Ω—è –≤ –Ü–Ω—Ç–µ—Ä–Ω–µ—Ç—ñ.

–î–ª—è –ø–æ—á–∞—Ç–∫—É –æ–±'—î–∫—Ç RegEx –ø–æ—Ç—Ä—ñ–±–Ω–æ —Å–∫–æ–º–ø—ñ–ª—é–≤–∞—Ç–∏ –∑ —à–∞–±–ª–æ–Ω–æ–º –ø–æ—à—É–∫—É –∑–∞ –¥–æ–ø–æ–º–æ–≥–æ—é –º–µ—Ç–æ–¥—É ``compile``, –ø–µ—Ä—à –Ω—ñ–∂ –π–æ–≥–æ –º–æ–∂–Ω–∞ –±—É–¥–µ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏.

::

    var regex = RegEx.new()
    regex.compile("\\w-(\\d+)")

–®–∞–±–ª–æ–Ω –ø–æ—à—É–∫—É —Å–ø–æ—á–∞—Ç–∫—É –ø–æ—Ç—Ä—ñ–±–Ω–æ –µ–∫—Ä–∞–Ω—É–≤–∞—Ç–∏ –¥–ª—è GDScript, –ø–µ—Ä—à –Ω—ñ–∂ –π–æ–≥–æ –µ–∫—Ä–∞–Ω—É–≤–∞—Ç–∏ –¥–ª—è –≤–∏—Ä–∞–∑—É. –ù–∞–ø—Ä–∏–∫–ª–∞–¥, ``compile("\\d+")`` –±—É–¥–µ –ø—Ä–æ—á–∏—Ç–∞–Ω–æ RegEx —è–∫ ``\d+``. –ê–Ω–∞–ª–æ–≥—ñ—á–Ω–æ, ``compile("\"(?:\\\\.|[^\"])*\"")`` –±—É–¥–µ —á–∏—Ç–∞—Ç–∏—Å—è —è–∫ ``"(?:\\.|[^"])*"``. –£ GDScript —Ç–∞–∫–æ–∂ –º–æ–∂–Ω–∞ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ –Ω–µ–æ–±—Ä–æ–±–ª–µ–Ω—ñ —Ä—è–¥–∫–æ–≤—ñ –ª—ñ—Ç–µ—Ä–∞–ª–∏ (r-—Ä—è–¥–∫–∏). –ù–∞–ø—Ä–∏–∫–ª–∞–¥, ``compile(r'"(?:\\.|[^"])*"')`` –±—É–¥–µ –ø—Ä–æ—á–∏—Ç–∞–Ω–æ —Ç–∞–∫ —Å–∞–º–æ.

–í–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—é—á–∏ :ref:`search()<class_RegEx_method_search>`, –≤–∏ –º–æ–∂–µ—Ç–µ –∑–Ω–∞–π—Ç–∏ —à–∞–±–ª–æ–Ω —É –∑–∞–¥–∞–Ω–æ–º—É —Ç–µ–∫—Å—Ç—ñ. –Ø–∫—â–æ —à–∞–±–ª–æ–Ω –∑–Ω–∞–π–¥–µ–Ω–æ, –ø–æ–≤–µ—Ä—Ç–∞—î—Ç—å—Å—è :ref:`RegExMatch<class_RegExMatch>`, —ñ –≤–∏ –º–æ–∂–µ—Ç–µ –æ—Ç—Ä–∏–º–∞—Ç–∏ –¥–µ—Ç–∞–ª—å–Ω—É —ñ–Ω—Ñ–æ—Ä–º–∞—Ü—ñ—é –ø—Ä–æ —Ä–µ–∑—É–ª—å—Ç–∞—Ç–∏ –∑–∞ –¥–æ–ø–æ–º–æ–≥–æ—é —Ç–∞–∫–∏—Ö –º–µ—Ç–æ–¥—ñ–≤, —è–∫ :ref:`RegExMatch.get_string()<class_RegExMatch_method_get_string>` —Ç–∞ :ref:`RegExMatch.get_start()<class_RegExMatch_method_get_start>`.

    var result = regex.search("abc n-0123")

    print(result.get_string()) # –í–∏–≤–æ–¥–∏—Ç—å "n-0123"

–†–µ–∑—É–ª—å—Ç–∞—Ç–∏ –∑–∞—Ö–æ–ø–ª–µ–Ω–Ω—è –≥—Ä—É–ø ``()`` –º–æ–∂–Ω–∞ –æ—Ç—Ä–∏–º–∞—Ç–∏, –ø–µ—Ä–µ–¥–∞–≤—à–∏ –Ω–æ–º–µ—Ä –≥—Ä—É–ø–∏ —Ä—ñ–∑–Ω–∏–º –º–µ—Ç–æ–¥–∞–º —É :ref:`RegExMatch<class_RegExMatch>`. –ì—Ä—É–ø–∞ 0 —î –≥—Ä—É–ø–æ—é –∑–∞ –∑–∞–º–æ–≤—á—É–≤–∞–Ω–Ω—è–º —ñ –∑–∞–≤–∂–¥–∏ –ø–æ—Å–∏–ª–∞—Ç–∏–º–µ—Ç—å—Å—è –Ω–∞ –≤–µ—Å—å —à–∞–±–ª–æ–Ω. –£ –Ω–∞–≤–µ–¥–µ–Ω–æ–º—É –≤–∏—â–µ –ø—Ä–∏–∫–ª–∞–¥—ñ –≤–∏–∫–ª–∏–∫ ``result.get_string(1)`` –≤–∏–¥–∞—Å—Ç—å –≤–∞–º ``0123``.

–¶—è –≤–µ—Ä—Å—ñ—è RegEx —Ç–∞–∫–æ–∂ –ø—ñ–¥—Ç—Ä–∏–º—É—î —ñ–º–µ–Ω–æ–≤–∞–Ω—ñ –≥—Ä—É–ø–∏ –∑–∞—Ö–æ–ø–ª–µ–Ω–Ω—è, —ñ–º–µ–Ω–∞ —è–∫–∏—Ö –º–æ–∂–Ω–∞ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ –¥–ª—è –æ—Ç—Ä–∏–º–∞–Ω–Ω—è —Ä–µ–∑—É–ª—å—Ç–∞—Ç—ñ–≤. –Ø–∫—â–æ –¥–≤—ñ –∞–±–æ –±—ñ–ª—å—à–µ –≥—Ä—É–ø –º–∞—é—Ç—å –æ–¥–Ω–∞–∫–æ–≤—É –Ω–∞–∑–≤—É, –Ω–∞–∑–≤–∞ –ø–æ—Å–∏–ª–∞—Ç–∏–º–µ—Ç—å—Å—è –ª–∏—à–µ –Ω–∞ –ø–µ—Ä—à—É –∑ –Ω–∏—Ö, —è–∫–∞ –≤—ñ–¥–ø–æ–≤—ñ–¥–∞—î.

–Ø–∫—â–æ –≤–∞–º –ø–æ—Ç—Ä—ñ–±–Ω–æ –æ–±—Ä–æ–±–∏—Ç–∏ –∫—ñ–ª—å–∫–∞ —Ä–µ–∑—É–ª—å—Ç–∞—Ç—ñ–≤, :ref:`search_all()<class_RegEx_method_search_all>` –≥–µ–Ω–µ—Ä—É—î —Å–ø–∏—Å–æ–∫ —É—Å—ñ—Ö –Ω–µ–ø–µ—Ä–µ–∫—Ä–∏–≤–∞—é—á–∏—Ö—Å—è —Ä–µ–∑—É–ª—å—Ç–∞—Ç—ñ–≤. –î–ª—è –∑—Ä—É—á–Ω–æ—Å—Ç—ñ —Ü–µ –º–æ–∂–Ω–∞ –ø–æ—î–¥–Ω–∞—Ç–∏ –∑ —Ü–∏–∫–ª–æ–º ``for``.
