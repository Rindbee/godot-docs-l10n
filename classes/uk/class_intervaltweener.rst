IntervalTweener
===============

**Успадковує:** :ref:`Tweener<class_Tweener>` **<** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

Створює інтервал свічки в анімації :ref:`Tween<class_Tween>`.

.. rst-class:: classref-introduction-group

Опис
--------

**IntervalTweener** використовується для затримок в послідовності висування. Див. :ref:`Tween.tween_interval()<class_Tween_method_tween_interval>` для отримання додаткової інформації про використання.

\ **Note:** :ref:`Tween.tween_interval()<class_Tween_method_tween_interval>` - єдиний правильний спосіб створення **IntervalTweener**. Будь-який **IntervalTweener** створений вручну не буде функціонувати правильно.
