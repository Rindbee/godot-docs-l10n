VSplitContainer
===============

**Успадковує:** :ref:`SplitContainer<class_SplitContainer>` **<** :ref:`Container<class_Container>` **<** :ref:`Control<class_Control>` **<** :ref:`CanvasItem<class_CanvasItem>` **<** :ref:`Node<class_Node>` **<** :ref:`Object<class_Object>`

Контейнер, що розщеплює два дочірні елементи, вертикально і забезпечує зчеплення для регулювання коефіцієнта розщеплення.

.. rst-class:: classref-introduction-group

Опис
--------

Контейнер, який приймає тільки два дочірні елементи, потім влаштовує їх вертикально і створює дивізор між ними. Дивізор може бути перетягований навколо, щоб змінити співвідношення розмірів між контрольами дитини.
