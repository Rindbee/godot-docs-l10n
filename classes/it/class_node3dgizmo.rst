Node3DGizmo
===========

**Eredita:** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

**Ereditato da:** :ref:`EditorNode3DGizmo<class_EditorNode3DGizmo>`

Classe astratta per esporre i gizmo dell'editor per :ref:`Node3D<class_Node3D>`.

.. rst-class:: classref-introduction-group

Descrizione
----------------------

Questa classe astratta aiuta a connettere la scena :ref:`Node3D<class_Node3D>` con la classe :ref:`EditorNode3DGizmo<class_EditorNode3DGizmo>` specifica dell'editor.

\ **Node3DGizmo** di per sé non ha API esposte, fai riferimento a :ref:`Node3D.add_gizmo()<class_Node3D_method_add_gizmo>` e passagli un'istanza di :ref:`EditorNode3DGizmo<class_EditorNode3DGizmo>`.
