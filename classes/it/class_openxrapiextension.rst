OpenXRAPIExtension
==================

**Eredita:** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

Rende disponibile l'API di OpenXR per GDExtension.

.. rst-class:: classref-introduction-group

Descrizione
----------------------

**OpenXRAPIExtension** rende OpenXR disponibile per GDExtension. Fornisce l'API di OpenXR a GDExtension tramite il metodo :ref:`get_instance_proc_addr()<class_OpenXRAPIExtension_method_get_instance_proc_addr>` e l'istanza OpenXR tramite :ref:`get_instance()<class_OpenXRAPIExtension_method_get_instance>`.

Fornisce inoltre metodi per interrogare lo stato di inizializzazione di OpenXR e metodi di supporto per semplificare l'utilizzo dell'API con GDExtension.
