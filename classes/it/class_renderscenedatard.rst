RenderSceneDataRD
=================

**Eredita:** :ref:`RenderSceneData<class_RenderSceneData>` **<** :ref:`Object<class_Object>`

Implementazione dei dati di scena di rendering per i renderer basati su RenderingDevice.

.. rst-class:: classref-introduction-group

Descrizione
----------------------

L'oggetto contiene i dati di scena relativi al rendering di un singolo frame di una viewport.

\ **Nota:** Questo è un oggetto interno nel server di rendering, non istanziarlo da script.
