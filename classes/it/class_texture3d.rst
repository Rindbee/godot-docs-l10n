Descrizione

Classe di base per :ref:`ImageTexture3D<class_ImageTexture3D>` e :ref:`CompressedTexture3D<class_CompressedTexture3D>`. Non puÃ² essere utilizzata direttamente, ma contiene tutte le funzioni necessarie per accedere ai tipi di risorse derivati. **Texture3D** Ã¨ la classe base per tutti i tipi di texture tridimensionali.
