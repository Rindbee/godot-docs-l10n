Descrizione
----------------------

Classe di base astratta per le barre di scorrimento, solitamente utilizzata per navigare attraverso contenuti che si estendono oltre l'area visibile di un controllo. Le barre di scorrimento sono controlli basati su :ref:`Range<class_Range>`.
