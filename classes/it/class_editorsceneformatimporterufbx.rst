EditorSceneFormatImporterUFBX
=============================

**Sperimentale:** This class may be changed or removed in future versions.

**Eredita:** :ref:`EditorSceneFormatImporter<class_EditorSceneFormatImporter>` **<** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

Importa file FBX attraverso la libreria ufbx.

.. rst-class:: classref-introduction-group

Descrizione
----------------------

EditorSceneFormatImporterUFBX è progettato per caricare file FBX e supporta sia file binari sia file FBX in ASCII dalla versione 3000 in poi. Questa classe supporta vari tipi di oggetti 3D come mesh, skin, forme di fusione, materiali e informazioni di rigging. La classe cerca di essere pari in funzionalità con l'SDK ufficiale di FBX e supporta le specifiche di FBX 7.4.
