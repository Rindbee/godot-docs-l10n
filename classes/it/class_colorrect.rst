ColorRect
=========

**Eredita:** :ref:`Control<class_Control>` **<** :ref:`CanvasItem<class_CanvasItem>` **<** :ref:`Node<class_Node>` **<** :ref:`Object<class_Object>`

Un controllo che visualizza un rettangolo di colore pieno.

.. rst-class:: classref-introduction-group

Descrizione
----------------------

Visualizza un rettangolo riempito con un colore (:ref:`color<class_ColorRect_property_color>`) pieno. Se hai bisogno di visualizzare solo il bordo, considera invece usare un :ref:`Panel<class_Panel>`.
