Un nodo di visual shader per il parametro di shader (uniforme) di tipo :ref:`Texture2DArray<class_Texture2DArray>`.

.. rst-class:: classref-introduction-group

Descrizione
----------------------

Questo parametro consente di fornire un insieme di texture per lo shader. È possibile utilizzare :ref:`VisualShaderNodeTexture2DArray<class_VisualShaderNodeTexture2DArray>` per estrarre le texture dall'array.
