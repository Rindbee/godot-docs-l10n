PhysicsServer2DExtension
========================

**Eredita:** :ref:`PhysicsServer2D<class_PhysicsServer2D>` **<** :ref:`Object<class_Object>`

Fornisce metodi virtuali che possono essere sovrascritti per creare implementazioni di :ref:`PhysicsServer2D<class_PhysicsServer2D>` personalizzate.

.. rst-class:: classref-introduction-group

Descrizione
----------------------

Questa classe estende :ref:`PhysicsServer2D<class_PhysicsServer2D>` fornendo metodi virtuali aggiuntivi che possono essere sovrascritti. Quando questi metodi sono sovrascritti, saranno chiamati al posto dei metodi interni del server di fisica.

Progettato per l'uso con GDExtension per creare implementazioni personalizzate di :ref:`PhysicsServer2D<class_PhysicsServer2D>`.
