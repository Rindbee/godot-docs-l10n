說明
----

在著色器語言中翻譯為 ``faceforward(N, I, Nref)``\ 。該函式有三個向量參數。\ ``N``\ ，定向向量，\ ``I``\ ，入射向量，以及\ ``Nref``\ ，參考向量。如果 ``I`` 和 ``Nref`` 的點積小於零，返回值為 ``N``\ 。否則，將返回 ``-N``\ 。
