Lightmapper
===========

**Наследует:** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

**Наследуется от:** :ref:`LightmapperRD<class_LightmapperRD>`

Абстрактный класс, расширенный lightmappers, для использования в :ref:`LightmapGI<class_LightmapGI>`.

.. rst-class:: classref-introduction-group

Описание
----------------

Этот класс должен быть расширен пользовательскими классами lightmapper. Lightmapper затем можно использовать с :ref:`LightmapGI<class_LightmapGI>` для обеспечения быстрого запеченного глобального освещения в 3D.

Godot содержит встроенный lightmapper на основе GPU :ref:`LightmapperRD<class_LightmapperRD>`, который использует вычислительные шейдеры, но пользовательские lightmapper могут быть реализованы с помощью модулей C++.
