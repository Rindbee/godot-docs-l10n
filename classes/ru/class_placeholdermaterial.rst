PlaceholderMaterial
===================

**Наследует:** :ref:`Material<class_Material>` **<** :ref:`Resource<class_Resource>` **<** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

Класс-заполнитель для материала.

.. rst-class:: classref-introduction-group

Описание
----------------

Этот класс используется при загрузке проекта, использующего подкласс :ref:`Material<class_Material>` в 2 условиях:

- При запуске проекта, экспортированного в режиме выделенного сервера, сохраняются только размеры текстуры (так как они могут быть использованы для игровых целей или позиционирования других элементов). Это позволяет значительно уменьшить размер экспортируемого PCK.

- Когда этот подкласс отсутствует из-за использования другой версии движка или сборки (например, отключены модули).
