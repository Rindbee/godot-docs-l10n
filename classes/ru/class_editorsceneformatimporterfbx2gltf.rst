EditorSceneFormatImporterFBX2GLTF
=================================

**Наследует:** :ref:`EditorSceneFormatImporter<class_EditorSceneFormatImporter>` **<** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

Импортер для формата файла сцены ``.fbx``.

.. rst-class:: classref-introduction-group

Описание
----------------

Импортирует сцены Autodesk FBX 3D путем преобразования их в glTF 2.0 с помощью инструмента командной строки FBX2glTF.

Расположение двоичного файла FBX2glTF задается с помощью настройки редактора :ref:`EditorSettings.filesystem/import/fbx/fbx2gltf_path<class_EditorSettings_property_filesystem/import/fbx/fbx2gltf_path>`.

Этот импортер используется только в том случае, если :ref:`ProjectSettings.filesystem/import/fbx2gltf/enabled<class_ProjectSettings_property_filesystem/import/fbx2gltf/enabled>` установлен на ``true``.
