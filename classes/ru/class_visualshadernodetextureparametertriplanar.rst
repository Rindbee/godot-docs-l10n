VisualShaderNodeTextureParameterTriplanar
=========================================

Выполняет равномерный поиск текстуры с помощью трипланара в графе визуального шейдера.

.. rst-class:: classref-introduction-group

Описание
----------------

Выполняет операцию поиска по текстуре, предоставленной в качестве однородной для шейдера, с поддержкой трипланарного отображения.
