CodeEdit
========

**Наследует:** :ref:`TextEdit<class_TextEdit>` **<** :ref:`Control<class_Control>` **<** :ref:`CanvasItem<class_CanvasItem>` **<** :ref:`Node<class_Node>` **<** :ref:`Object<class_Object>`

Многострочный текстовый редактор, предназначенный для редактирования кода.

.. rst-class:: classref-introduction-group

Описание
----------------

CodeEdit — это специализированный :ref:`TextEdit<class_TextEdit>`, разработанный для редактирования файлов с простым текстовым кодом. Он обладает множеством функций, обычно встречающихся в редакторах кода, таких как номера строк, перенос строк, автодополнение кода, управление отступами и управление строками/комментариями.

\ **Примечание:** Независимо от локали, **CodeEdit** по умолчанию всегда будет использовать направление текста слева направо для правильного отображения исходного кода.
