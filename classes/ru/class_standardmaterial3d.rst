StandardMaterial3D
==================

**Наследует:** :ref:`BaseMaterial3D<class_BaseMaterial3D>` **<** :ref:`Material<class_Material>` **<** :ref:`Resource<class_Resource>` **<** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

Материал PBR (физически обоснованный рендеринг) для использования на 3D-объектах.

.. rst-class:: classref-introduction-group

Описание
----------------

Свойства **StandardMaterial3D** унаследованы от :ref:`BaseMaterial3D<class_BaseMaterial3D>`. **StandardMaterial3D** использует отдельные текстуры для карт окклюзии окружающего пространства, шероховатости и металличности. Чтобы использовать одну карту ORM для всех 3 текстур, используйте вместо этого :ref:`ORMMaterial3D<class_ORMMaterial3D>`.
