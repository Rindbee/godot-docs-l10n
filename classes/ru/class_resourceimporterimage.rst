ResourceImporterImage
=====================

**Наследует:** :ref:`ResourceImporter<class_ResourceImporter>` **<** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

Импортирует изображение для использования в скриптах, без возможности рендеринга.

.. rst-class:: classref-introduction-group

Описание
----------------

Этот импортер импортирует ресурсы :ref:`Image<class_Image>`, а не :ref:`CompressedTexture2D<class_CompressedTexture2D>`. Если вам нужно визуализировать изображение в 2D или 3D, используйте вместо этого :ref:`ResourceImporterTexture<class_ResourceImporterTexture>`.
