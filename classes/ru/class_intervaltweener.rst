IntervalTweener
===============

**Наследует:** :ref:`Tweener<class_Tweener>` **<** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

Создает интервал простоя в анимации :ref:`Tween<class_Tween>`.

.. rst-class:: classref-introduction-group

Описание
----------------

**IntervalTweener** используется для создания задержек в последовательности промежуточных кадров. Для получения дополнительной информации об использовании см. :ref:`Tween.tween_interval()<class_Tween_method_tween_interval>`.

\ **Примечание:** :ref:`Tween.tween_interval()<class_Tween_method_tween_interval>` — единственный правильный способ создания **IntervalTweener**. Любой **IntervalTweener**, созданный вручную, не будет работать правильно.
