Описание
----------------

EditorSceneFormatImporterUFBX предназначен для загрузки файлов FBX и поддерживает как двоичные, так и ASCII файлы FBX, начиная с версии 3000. Этот класс поддерживает различные типы 3D-объектов, такие как сетки, оболочки, смешанные формы, материалы и информацию об оснастке. Класс нацелен на паритет функций с официальным FBX SDK и поддерживает спецификации FBX 7.4.
