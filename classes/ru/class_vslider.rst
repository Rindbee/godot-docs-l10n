Вертикальный ползунок, перемещающийся снизу (мин) вверх (макс).

.. rst-class:: classref-introduction-group

Описание
----------------

Вертикальный ползунок, используемый для регулировки значения путем перемещения захвата вдоль вертикальной оси. Это элемент управления на основе :ref:`Range<class_Range>` и идет снизу (мин.) вверх (макс.). Обратите внимание, что это направление противоположно направлению :ref:`VScrollBar<class_VScrollBar>`.
