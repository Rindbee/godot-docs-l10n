Описание
----------------

Это :ref:`MultiplayerAPI.multiplayer_peer<class_MultiplayerAPI_property_multiplayer_peer>` по умолчанию для :ref:`Node.multiplayer<class_Node_property_multiplayer>`. Он имитирует поведение сервера без подключенных пиров.

Это означает, что :ref:`SceneTree<class_SceneTree>` будет действовать как многопользовательский по умолчанию. Вызовы :ref:`MultiplayerAPI.is_server()<class_MultiplayerAPI_method_is_server>` вернут ``true``, а вызовы :ref:`MultiplayerAPI.get_unique_id()<class_MultiplayerAPI_method_get_unique_id>` вернут :ref:`MultiplayerPeer.TARGET_PEER_SERVER<class_MultiplayerPeer_constant_TARGET_PEER_SERVER>`.
