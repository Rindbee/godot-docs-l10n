LightmapProbe
=============

**Наследует:** :ref:`Node3D<class_Node3D>` **<** :ref:`Node<class_Node>` **<** :ref:`Object<class_Object>`

Представляет собой одиночный вручную размещенный зонд для динамического освещения объекта с помощью :ref:`LightmapGI<class_LightmapGI>`.

.. rst-class:: classref-introduction-group

Описание
----------------

**LightmapProbe** представляет положение одного вручную размещенного зонда для динамического освещения объекта с помощью :ref:`LightmapGI<class_LightmapGI>`. Зонды карты освещения влияют на освещение узлов, производных от :ref:`GeometryInstance3D<class_GeometryInstance3D>`, у которых :ref:`GeometryInstance3D.gi_mode<class_GeometryInstance3D_property_gi_mode>` установлен на :ref:`GeometryInstance3D.GI_MODE_DYNAMIC<class_GeometryInstance3D_constant_GI_MODE_DYNAMIC>`.

Обычно зонды :ref:`LightmapGI<class_LightmapGI>` размещаются автоматически путем установки :ref:`LightmapGI.generate_probes_subdiv<class_LightmapGI_property_generate_probes_subdiv>` на значение, отличное от :ref:`LightmapGI.GENERATE_PROBES_DISABLED<class_LightmapGI_constant_GENERATE_PROBES_DISABLED>`. Создав узлы **LightmapProbe** перед запеканием карт освещения, вы можете добавить больше зондов в определенных областях для большей детализации или отключить автоматическую генерацию и вместо этого полагаться только на зонды, размещенные вручную.

\ **Примечание:** Узлы **LightmapProbe**, размещенные после запекания карт освещения, игнорируются динамическими объектами. После создания или изменения **LightmapProbe** необходимо заново запечь карты освещения, чтобы зонды были эффективными.
