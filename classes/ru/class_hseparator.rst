HSeparator

.. rst-class:: classref-introduction-group

Описание
----------------

Горизонтальный разделитель, используемый для разделения других элементов управления, расположенных **вертикально**. **HSeparator** является чисто визуальным и обычно отображается как :ref:`StyleBoxLine<class_StyleBoxLine>`.
