–ê–ª—å—Ç–µ—Ä–Ω–∞—Ç–∏–≤—É –±–æ–ª–µ–µ –≤—ã—Å–æ–∫–æ–≥–æ —É—Ä–æ–≤–Ω—è —Å–º. –≤ —É–∑–ª–µ :ref:`HTTPRequest<class_HTTPRequest>`.

\ **–ü—Ä–∏–º–µ—á–∞–Ω–∏–µ:** –≠—Ç–æ–º—É –∫–ª–∏–µ–Ω—Ç—É –¥–æ—Å—Ç–∞—Ç–æ—á–Ω–æ –ø–æ–¥–∫–ª—é—á–∏—Ç—å—Å—è –∫ —Ö–æ—Å—Ç—É –æ–¥–∏–Ω —Ä–∞–∑ (—Å–º. :ref:`connect_to_host()<class_HTTPClient_method_connect_to_host>`) –¥–ª—è –æ—Ç–ø—Ä–∞–≤–∫–∏ –Ω–µ—Å–∫–æ–ª—å–∫–∏—Ö –∑–∞–ø—Ä–æ—Å–æ–≤. –ü–æ —ç—Ç–æ–π –ø—Ä–∏—á–∏–Ω–µ –º–µ—Ç–æ–¥—ã, –ø—Ä–∏–Ω–∏–º–∞—é—â–∏–µ URL-–∞–¥—Ä–µ—Å–∞, –æ–±—ã—á–Ω–æ –±–µ—Ä—É—Ç —Ç–æ–ª—å–∫–æ —á–∞—Å—Ç—å –ø–æ—Å–ª–µ —Ö–æ—Å—Ç–∞, –∞ –Ω–µ –ø–æ–ª–Ω—ã–π URL-–∞–¥—Ä–µ—Å, –ø–æ—Å–∫–æ–ª—å–∫—É –∫–ª–∏–µ–Ω—Ç —É–∂–µ –ø–æ–¥–∫–ª—é—á–µ–Ω –∫ —Ö–æ—Å—Ç—É. –ü–æ–ª–Ω—ã–π –ø—Ä–∏–º–µ—Ä –∏ –∏–Ω—Å—Ç—Ä—É–∫—Ü–∏–∏ –ø–æ –Ω–∞—á–∞–ª—É —Ä–∞–±–æ—Ç—ã —Å–º. –≤ :ref:`request()<class_HTTPClient_method_request>`.

\ **HTTPClient** —Å–ª–µ–¥—É–µ—Ç –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –ø–æ–≤—Ç–æ—Ä–Ω–æ –º–µ–∂–¥—É –Ω–µ—Å–∫–æ–ª—å–∫–∏–º–∏ –∑–∞–ø—Ä–æ—Å–∞–º–∏ –∏–ª–∏ –¥–ª—è –ø–æ–¥–∫–ª—é—á–µ–Ω–∏—è –∫ —Ä–∞–∑–Ω—ã–º —Ö–æ—Å—Ç–∞–º –≤–º–µ—Å—Ç–æ —Å–æ–∑–¥–∞–Ω–∏—è –æ–¥–Ω–æ–≥–æ –∫–ª–∏–µ–Ω—Ç–∞ –Ω–∞ –∫–∞–∂–¥—ã–π –∑–∞–ø—Ä–æ—Å. –ü–æ–¥–¥–µ—Ä–∂–∏–≤–∞–µ—Ç –ø—Ä–æ—Ç–æ–∫–æ–ª Transport Layer Security (TLS), –≤–∫–ª—é—á–∞—è –ø—Ä–æ–≤–µ—Ä–∫—É —Å–µ—Ä—Ç–∏—Ñ–∏–∫–∞—Ç–∞ —Å–µ—Ä–≤–µ—Ä–∞. –ö–æ–¥—ã —Å—Ç–∞—Ç—É—Å–∞ HTTP –≤ –¥–∏–∞–ø–∞–∑–æ–Ω–µ 2xx —É–∫–∞–∑—ã–≤–∞—é—Ç –Ω–∞ —É—Å–ø–µ—à–Ω–æ–µ –≤—ã–ø–æ–ª–Ω–µ–Ω–∏–µ, 3xx ‚Äî –Ω–∞ –ø–µ—Ä–µ–Ω–∞–ø—Ä–∞–≤–ª–µ–Ω–∏–µ (—Ç. –µ. ¬´–ø–æ–ø—Ä–æ–±—É–π—Ç–µ –µ—â–µ —Ä–∞–∑, –Ω–æ –∑–¥–µ—Å—å¬ª), 4xx ‚Äî –Ω–∞ –æ—à–∏–±–∫—É –≤ –∑–∞–ø—Ä–æ—Å–µ –∏ 5xx ‚Äî –Ω–∞ –æ—à–∏–±–∫—É –Ω–∞ —Å—Ç–æ—Ä–æ–Ω–µ —Å–µ—Ä–≤–µ—Ä–∞.

–î–ª—è –ø–æ–ª—É—á–µ–Ω–∏—è –¥–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω–æ–π –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏–∏ –æ HTTP —Å–º. `MDN's documentation on HTTP <https://developer.mozilla.org/en-US/docs/Web/HTTP>`__ (–∏–ª–∏ –ø—Ä–æ—á–∏—Ç–∞–π—Ç–µ `RFC 2616 <https://tools.ietf.org/html/rfc2616>`__ —á—Ç–æ–±—ã –ø–æ–ª—É—á–∏—Ç—å –ø—Ä—è–º–æ –∏–∑ –∏—Å—Ç–æ—á–Ω–∏–∫–∞).

\ **–ü—Ä–∏–º–µ—á–∞–Ω–∏–µ:** –ü—Ä–∏ —ç–∫—Å–ø–æ—Ä—Ç–µ –Ω–∞ Android —É–±–µ–¥–∏—Ç–µ—Å—å, —á—Ç–æ –≤ –Ω–∞—Å—Ç—Ä–æ–π–∫–∞—Ö —ç–∫—Å–ø–æ—Ä—Ç–∞ Android –≤–∫–ª—é—á–µ–Ω–æ —Ä–∞–∑—Ä–µ—à–µ–Ω–∏–µ ``INTERNET`` –ø–µ—Ä–µ–¥ —ç–∫—Å–ø–æ—Ä—Ç–æ–º –ø—Ä–æ–µ–∫—Ç–∞ –∏–ª–∏ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ–º —Ä–∞–∑–≤–µ—Ä—Ç—ã–≤–∞–Ω–∏—è –≤ –æ–¥–∏–Ω –∫–ª–∏–∫. –í –ø—Ä–æ—Ç–∏–≤–Ω–æ–º —Å–ª—É—á–∞–µ Android –±—É–¥–µ—Ç –±–ª–æ–∫–∏—Ä–æ–≤–∞—Ç—å –ª—é–±—ã–µ —Å–µ—Ç–µ–≤—ã–µ —Å–æ–µ–¥–∏–Ω–µ–Ω–∏—è.

\ **–ü—Ä–∏–º–µ—á–∞–Ω–∏–µ:** –†–µ–∫–æ–º–µ–Ω–¥—É–µ—Ç—Å—è –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å —Ç—Ä–∞–Ω—Å–ø–æ—Ä—Ç–Ω–æ–µ —à–∏—Ñ—Ä–æ–≤–∞–Ω–∏–µ (TLS) –∏ –∏–∑–±–µ–≥–∞—Ç—å –ø–µ—Ä–µ–¥–∞—á–∏ –∫–æ–Ω—Ñ–∏–¥–µ–Ω—Ü–∏–∞–ª—å–Ω–æ–π –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏–∏ (–Ω–∞–ø—Ä–∏–º–µ—Ä, —É—á—ë—Ç–Ω—ã—Ö –¥–∞–Ω–Ω—ã—Ö) –≤ –ø–∞—Ä–∞–º–µ—Ç—Ä–∞—Ö URL HTTP GET. –í–º–µ—Å—Ç–æ —ç—Ç–æ–≥–æ —Ä–∞—Å—Å–º–æ—Ç—Ä–∏—Ç–µ –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç—å –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è HTTP POST-–∑–∞–ø—Ä–æ—Å–æ–≤ –∏–ª–∏ HTTP-–∑–∞–≥–æ–ª–æ–≤–∫–æ–≤ –¥–ª—è –ø–µ—Ä–µ–¥–∞—á–∏ —Ç–∞–∫–æ–π –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏–∏.

\ **–ü—Ä–∏–º–µ—á–∞–Ω–∏–µ:** –ü—Ä–∏ –≤—ã–ø–æ–ª–Ω–µ–Ω–∏–∏ HTTP-–∑–∞–ø—Ä–æ—Å–æ–≤ –∏–∑ –ø—Ä–æ–µ–∫—Ç–∞, —ç–∫—Å–ø–æ—Ä—Ç–∏—Ä–æ–≤–∞–Ω–Ω–æ–≥–æ –≤ –ò–Ω—Ç–µ—Ä–Ω–µ—Ç, –ø–æ–º–Ω–∏—Ç–µ, —á—Ç–æ —É–¥–∞–ª–µ–Ω–Ω—ã–π —Å–µ—Ä–≤–µ—Ä –º–æ–∂–µ—Ç –Ω–µ —Ä–∞–∑—Ä–µ—à–∞—Ç—å –∑–∞–ø—Ä–æ—Å—ã –∏–∑ –≤–Ω–µ—à–Ω–∏—Ö –∏—Å—Ç–æ—á–Ω–∏–∫–æ–≤ –∏–∑-–∑–∞ `CORS <https://developer.mozilla.org/en-US/docs/Web/HTTP/CORS>`__. –ï—Å–ª–∏ –≤—ã —Ä–∞–∑–º–µ—â–∞–µ—Ç–µ —Ä–∞—Å—Å–º–∞—Ç—Ä–∏–≤–∞–µ–º—ã–π —Å–µ—Ä–≤–µ—Ä, –≤–∞–º —Å–ª–µ–¥—É–µ—Ç –∏–∑–º–µ–Ω–∏—Ç—å –µ–≥–æ –±—ç–∫—ç–Ω–¥, —á—Ç–æ–±—ã —Ä–∞–∑—Ä–µ—à–∏—Ç—å –∑–∞–ø—Ä–æ—Å—ã –∏–∑ –≤–Ω–µ—à–Ω–∏—Ö –∏—Å—Ç–æ—á–Ω–∏–∫–æ–≤, –¥–æ–±–∞–≤–∏–≤ HTTP-–∑–∞–≥–æ–ª–æ–≤–æ–∫ ``Access-Control-Allow-Origin: *``.
