TreeItem
========

**Наследует:** :ref:`Object<class_Object>`

Внутренний control для одного элемента внутри :ref:`Tree<class_Tree>`.

.. rst-class:: classref-introduction-group

Описание
----------------

Один элемент элемента управления :ref:`Tree<class_Tree>`. Он может содержать другие **TreeItem** в качестве дочерних элементов, что позволяет ему создавать иерархию. Он также может содержать текст и кнопки. **TreeItem** не является :ref:`Node<class_Node>`, он является внутренним для :ref:`Tree<class_Tree>`.

Чтобы создать **TreeItem**, используйте :ref:`Tree.create_item()<class_Tree_method_create_item>` или :ref:`create_child()<class_TreeItem_method_create_child>`. Чтобы удалить **TreeItem**, используйте :ref:`Object.free()<class_Object_method_free>`.

\ **Примечание:** Значения идентификаторов, используемые для кнопок, являются 32-битными, в отличие от :ref:`int<class_int>`, который всегда является 64-битным. Они идут от ``-2147483648`` до ``2147483647``.

.. _class_TreeItem_constant_CELL_MODE_STRING:

.. rst-class:: classref-enumeration-constant

:ref:`TreeCellMode<enum_TreeItem_TreeCellMode>` **CELL_MODE_STRING** = ``0``

Ячейка показывает строковую метку, опционально со значком. Если текст редактируемый, его можно редактировать с помощью :ref:`LineEdit<class_LineEdit>` или всплывающего окна :ref:`TextEdit<class_TextEdit>`, если используется :ref:`set_edit_multiline()<class_TreeItem_method_set_edit_multiline>`.

.. _class_TreeItem_constant_CELL_MODE_CHECK:

.. rst-class:: classref-enumeration-constant

:ref:`TreeCellMode<enum_TreeItem_TreeCellMode>` **CELL_MODE_CHECK** = ``1``

Ячейка показывает флажок, опционально с текстом и значком. Флажок можно нажать, отпустить или сделать неопределенным (через :ref:`set_indeterminate()<class_TreeItem_method_set_indeterminate>`). Флажок нельзя нажать, если ячейка не является редактируемой.

.. _class_TreeItem_constant_CELL_MODE_RANGE:

.. rst-class:: classref-enumeration-constant

:ref:`TreeCellMode<enum_TreeItem_TreeCellMode>` **CELL_MODE_RANGE** = ``2``

Ячейка показывает числовой диапазон. Если он доступен для редактирования, его можно редактировать с помощью ползунка диапазона. Используйте :ref:`set_range()<class_TreeItem_method_set_range>` для установки значения и :ref:`set_range_config()<class_TreeItem_method_set_range_config>` для настройки диапазона.

Эту ячейку также можно использовать в режиме раскрывающегося списка текста, когда вы назначаете текст с помощью :ref:`set_text()<class_TreeItem_method_set_text>`. Разделяйте параметры запятыми, например, ``"Option1,Option2,Option3"``.

.. _class_TreeItem_constant_CELL_MODE_ICON:

.. rst-class:: classref-enumeration-constant

:ref:`TreeCellMode<enum_TreeItem_TreeCellMode>` **CELL_MODE_ICON** = ``3``

Ячейка показывает значок. Его нельзя редактировать или отображать текст. Значок всегда находится в центре ячейки.

.. _class_TreeItem_constant_CELL_MODE_CUSTOM:

.. rst-class:: classref-enumeration-constant

:ref:`TreeCellMode<enum_TreeItem_TreeCellMode>` **CELL_MODE_CUSTOM** = ``4``

Ячейка отображается как нажимаемая кнопка. Она будет отображать стрелку, похожую на :ref:`OptionButton<class_OptionButton>`, но не имеет раскрывающегося списка (для этого вы можете использовать :ref:`CELL_MODE_RANGE<class_TreeItem_constant_CELL_MODE_RANGE>`). Нажатие кнопки испускает сигнал :ref:`Tree.item_edited<class_Tree_signal_item_edited>`. Кнопка по умолчанию плоская, вы можете использовать :ref:`set_custom_as_button()<class_TreeItem_method_set_custom_as_button>`, чтобы отобразить ее с :ref:`StyleBox<class_StyleBox>`.

Этот режим также поддерживает пользовательское рисование с помощью :ref:`set_custom_draw_callback()<class_TreeItem_method_set_custom_draw_callback>`.

.. _class_TreeItem_method_add_button:

.. rst-class:: classref-method

|void| **add_button**\ (\ column\: :ref:`int<class_int>`, button\: :ref:`Texture2D<class_Texture2D>`, id\: :ref:`int<class_int>` = -1, disabled\: :ref:`bool<class_bool>` = false, tooltip_text\: :ref:`String<class_String>` = "", description\: :ref:`String<class_String>` = ""\ ) :ref:`🔗<class_TreeItem_method_add_button>`

Добавляет кнопку с :ref:`Texture2D<class_Texture2D>` ``button`` в конец ячейки в столбце ``column``. Параметр ``id`` используется для идентификации кнопки в соответствующем сигнале :ref:`Tree.button_clicked<class_Tree_signal_button_clicked>` и может отличаться от индекса кнопки. Если не указан, используется следующий доступный индекс, который можно получить, вызвав :ref:`get_button_count()<class_TreeItem_method_get_button_count>` непосредственно перед этим методом. При желании кнопка может быть ``disable`` и иметь ``tooltip_text``. ``description`` используется в качестве описания кнопки для вспомогательных приложений.
