TextServerAdvanced
==================

**Наследует:** :ref:`TextServerExtension<class_TextServerExtension>` **<** :ref:`TextServer<class_TextServer>` **<** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

Расширенный текстовый сервер с поддержкой BiDi, сложной текстовой компоновки и контекстных функций OpenType. Используется в Godot по умолчанию.

.. rst-class:: classref-introduction-group

Описание
----------------

Реализация :ref:`TextServer<class_TextServer>`, которая использует HarfBuzz, ICU и SIL Graphite для поддержки BiDi, сложных текстовых макетов и контекстных функций OpenType. Это основной интерфейс :ref:`TextServer<class_TextServer>` Godot по умолчанию.
