AudioBusLayout
==============

**Наследует:** :ref:`Resource<class_Resource>` **<** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

Сохраняет информацию об аудио шинах.

.. rst-class:: classref-introduction-group

Описание
----------------

Сохраняет позицию, отключение звука, соло, обход, эффекты, позицию эффекта, громкость и соединения между шинами. См. :ref:`AudioServer<class_AudioServer>` для использования.
