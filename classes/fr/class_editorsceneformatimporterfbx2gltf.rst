EditorSceneFormatImporterFBX2GLTF
=================================

**Hérite de :** :ref:`EditorSceneFormatImporter<class_EditorSceneFormatImporter>` **<** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

Importeur pour le format de fichier de scène ``.fbx``.

.. rst-class:: classref-introduction-group

Description
-----------

Importe des scènes 3D FBX Autodesk en les convertissant en glTF 2.0 à l'aide de l'outil en ligne de commande FBX2glTF.

L'emplacement du binaire FBX2glTF est défini via le paramètre :ref:`EditorSettings.filesystem/import/fbx/fbx2gltf_path<class_EditorSettings_property_filesystem/import/fbx/fbx2gltf_path>`.

Cet importateur n'est utilisé que si :ref:`ProjectSettings.filesystem/import/fbx2gltf/enabled<class_ProjectSettings_property_filesystem/import/fbx2gltf/enabled>` est défini à ``true``.
