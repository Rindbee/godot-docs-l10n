VBoxContainer
=============

Un conteneur qui arrange ses contrôles enfant verticalement.

.. rst-class:: classref-introduction-group

Description
-----------

Une variante de :ref:`BoxContainer<class_BoxContainer>` qui ne peut arranger ses contrôles enfants que verticalement. Les contrôles enfants sont ré-organisés automatiquement lorsque leur taille minimale change.
