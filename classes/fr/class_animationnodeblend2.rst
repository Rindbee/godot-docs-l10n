===================

Mélange deux animations de façon linéaire à l'intérieur d'un :ref:`AnimationNodeBlendTree<class_AnimationNodeBlendTree>`.

.. rst-class:: classref-introduction-group

Description
-----------

Une ressource pour ajouter à un :ref:`AnimationNodeBlendTree<class_AnimationNodeBlendTree>`. Mélange deux animations linéairement selon la valeur de la quantité.

En général, la valeur du mélange devrait être dans l'intervalle ``[0.0, 1.0]``. Les valeurs en dehors de cette plage peuvent mélanger des animations amplifiées ou inversées, cependant, :ref:`AnimationNodeAdd2<class_AnimationNodeAdd2>` fonctionne mieux à cette fin.
