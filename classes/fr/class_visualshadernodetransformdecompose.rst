VisualShaderNodeTransformDecompose
==================================

**Hérite de :** :ref:`VisualShaderNode<class_VisualShaderNode>` **<** :ref:`Resource<class_Resource>` **<** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

Décompose une :ref:`Transform3D<class_Transform3D>` en quatre :ref:`Vector3<class_Vector3>` dans le graphe de shader visuel.

.. rst-class:: classref-introduction-group

Description
-----------

Takes a 4×4 transform matrix and decomposes it into four ``vec3`` values, one from each row of the matrix.
