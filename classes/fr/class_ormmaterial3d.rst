ORMMaterial3D
=============

**Hérite de :** :ref:`BaseMaterial3D<class_BaseMaterial3D>` **<** :ref:`Material<class_Material>` **<** :ref:`Resource<class_Resource>` **<** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

Un matériau PBR (Physically Based Rendering, litt. "Rendu basé sur la physique") à utiliser sur des objets 3D. Utilise une texture ORM.

.. rst-class:: classref-introduction-group

Description
-----------

Les propriétés de ORMMaterial3D sont héritées de :ref:`BaseMaterial3D<class_BaseMaterial3D>`. Contrairement à :ref:`StandardMaterial3D<class_StandardMaterial3D>`, ORMMaterial3D utilise une texture unique pour l'occlusion ambiante, la rugosité et les metallic maps, appelée texture ORM.
