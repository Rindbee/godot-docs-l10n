.. rst-class:: classref-introduction-group

Descripción
----------------------

Recurso :ref:`VideoStream<class_VideoStream>` que maneja el formato de vídeo `Ogg Theora <https://www.theora.org/>`__ con extensión ``.ogv``. El códec Theora se decodifica en la CPU.

\ **Nota:** Si bien los vídeos Ogg Theora también pueden tener una extensión ``.ogg``, tendrás que cambiar el nombre de la extensión a ``.ogv`` para usar esos vídeos dentro de Godot.
