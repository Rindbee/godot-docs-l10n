ORMMaterial3D
=============

**Hereda:** :ref:`BaseMaterial3D<class_BaseMaterial3D>` **<** :ref:`Material<class_Material>` **<** :ref:`Resource<class_Resource>` **<** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

Un material PBR (Physically Based Rendering) que se utiliza en objetos 3D. Utiliza una textura ORM.

.. rst-class:: classref-introduction-group

Descripción
----------------------

Las propiedades de ORMMaterial3D se heredan de :ref:`BaseMaterial3D<class_BaseMaterial3D>`. A diferencia de :ref:`StandardMaterial3D<class_StandardMaterial3D>`, ORMMaterial3D utiliza una sola textura para los mapas de oclusión ambiental, rugosidad y metálicos, conocida como textura ORM.
