Projection
==========

Una matriz de 44 para transformaciones proyectivas 3D.

.. rst-class:: classref-introduction-group

Descripci贸n
----------------------

A 44 matrix used for 3D projective transformations. It can represent transformations such as translation, rotation, scaling, shearing, and perspective division. It consists of four :ref:`Vector4<class_Vector4>` columns.

For purely linear transformations (translation, rotation, and scale), it is recommended to use :ref:`Transform3D<class_Transform3D>`, as it is more performant and requires less memory.

Used internally as :ref:`Camera3D<class_Camera3D>`'s projection matrix.
