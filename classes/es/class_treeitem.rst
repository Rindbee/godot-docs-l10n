.. _class_TreeItem:

TreeItem
========

**Hereda:** :ref:`Object<class_Object>`

Un control interno para un solo elemento dentro de :ref:`Tree<class_Tree>`.

.. rst-class:: classref-introduction-group

Descripción
----------------------

Un único elemento de un control :ref:`Tree<class_Tree>`. Puede contener otros **TreeItem**\ s como hijos, lo que le permite crear una jerarquía. También puede contener texto y botones. **TreeItem** no es un :ref:`Node<class_Node>`, es interno al :ref:`Tree<class_Tree>`.

Para crear un **TreeItem**, usa :ref:`Tree.create_item()<class_Tree_method_create_item>` o :ref:`create_child()<class_TreeItem_method_create_child>`. Para eliminar un **TreeItem**, usa :ref:`Object.free()<class_Object_method_free>`.

\ **Nota:** Los valores ID usados para los botones son de 32-bit, a diferencia de :ref:`int<class_int>` que siempre es de 64-bit. Van desde ``-2147483648`` a ``2147483647``.

Enumeraciones

.. _enum_TreeItem_TreeCellMode:

.. rst-class:: classref-enumeration

enum **TreeCellMode**: :ref:`🔗<enum_TreeItem_TreeCellMode>`

.. _class_TreeItem_constant_CELL_MODE_STRING:

.. rst-class:: classref-enumeration-constant

:ref:`TreeCellMode<enum_TreeItem_TreeCellMode>` **CELL_MODE_STRING** = ``0``

La celda muestra una etiqueta de texto, opcionalmente con un icono. Cuando es editable, el texto se puede editar usando un :ref:`LineEdit<class_LineEdit>`, o un popup de :ref:`TextEdit<class_TextEdit>` si se usa :ref:`set_edit_multiline()<class_TreeItem_method_set_edit_multiline>`.

.. _class_TreeItem_constant_CELL_MODE_CHECK:

.. rst-class:: classref-enumeration-constant

:ref:`TreeCellMode<enum_TreeItem_TreeCellMode>` **CELL_MODE_CHECK** = ``1``

La celda muestra una casilla de verificación, opcionalmente con texto y un icono. La casilla de verificación puede estar presionada, liberada o indeterminada (a través de :ref:`set_indeterminate()<class_TreeItem_method_set_indeterminate>`). La casilla de verificación no se puede hacer clic a menos que la celda sea editable.

.. _class_TreeItem_constant_CELL_MODE_RANGE:

.. rst-class:: classref-enumeration-constant

:ref:`TreeCellMode<enum_TreeItem_TreeCellMode>` **CELL_MODE_RANGE** = ``2``

La celda muestra un rango numérico. Cuando es editable, se puede editar usando un deslizador de rango. Usa :ref:`set_range()<class_TreeItem_method_set_range>` para establecer el valor y :ref:`set_range_config()<class_TreeItem_method_set_range_config>` para configurar el rango.

Esta celda también se puede usar en un modo de lista desplegable de texto cuando asignas un texto con :ref:`set_text()<class_TreeItem_method_set_text>`. Separa las opciones con una coma, por ejemplo, ``"Opción1,Opción2,Opción3"``.

.. _class_TreeItem_constant_CELL_MODE_ICON:

.. rst-class:: classref-enumeration-constant

:ref:`TreeCellMode<enum_TreeItem_TreeCellMode>` **CELL_MODE_ICON** = ``3``

La celda muestra un icono. No se puede editar ni mostrar texto. El icono siempre está centrado dentro de la celda.

.. _class_TreeItem_constant_CELL_MODE_CUSTOM:

.. rst-class:: classref-enumeration-constant

:ref:`TreeCellMode<enum_TreeItem_TreeCellMode>` **CELL_MODE_CUSTOM** = ``4``

La celda se muestra como un botón en el que se puede hacer clic. Mostrará una flecha similar a :ref:`OptionButton<class_OptionButton>`, pero no presenta una lista desplegable (para eso puedes usar :ref:`CELL_MODE_RANGE<class_TreeItem_constant_CELL_MODE_RANGE>`). Al hacer clic en el botón se emite la señal :ref:`Tree.item_edited<class_Tree_signal_item_edited>`. El botón es plano por defecto, puedes usar :ref:`set_custom_as_button()<class_TreeItem_method_set_custom_as_button>` para mostrarlo con un :ref:`StyleBox<class_StyleBox>`.

Este modo también admite el dibujo personalizado mediante :ref:`set_custom_draw_callback()<class_TreeItem_method_set_custom_draw_callback>`.

.. _class_TreeItem_method_add_button:

.. rst-class:: classref-method

|void| **add_button**\ (\ column\: :ref:`int<class_int>`, button\: :ref:`Texture2D<class_Texture2D>`, id\: :ref:`int<class_int>` = -1, disabled\: :ref:`bool<class_bool>` = false, tooltip_text\: :ref:`String<class_String>` = "", description\: :ref:`String<class_String>` = ""\ ) :ref:`🔗<class_TreeItem_method_add_button>`

Añade un botón con :ref:`Texture2D<class_Texture2D>` ``button`` al final de la celda en la columna ``column``. El ``id`` se usa para identificar el botón en la señal correspondiente :ref:`Tree.button_clicked<class_Tree_signal_button_clicked>` y puede ser diferente del índice de los botones. Si no se especifica, se utiliza el siguiente índice disponible, que puede recuperarse llamando a :ref:`get_button_count()<class_TreeItem_method_get_button_count>` inmediatamente antes de este método. Opcionalmente, el botón puede estar ``disabled`` y tener un ``tooltip_text``. ``description`` se utiliza como descripción del botón para las aplicaciones de asistencia.
