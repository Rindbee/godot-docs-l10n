StyleBoxEmpty
=============

**Hereda:** :ref:`StyleBox<class_StyleBox>` **<** :ref:`Resource<class_Resource>` **<** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

Una :ref:`StyleBox<class_StyleBox>` vacía (no muestra nada).

.. rst-class:: classref-introduction-group

Descripción
----------------------

An empty :ref:`StyleBox<class_StyleBox>` that can be used to display nothing instead of the default style (e.g. it can "disable" ``focus`` styles).
