Texture2DArrayRD
================

**Hereda:** :ref:`TextureLayeredRD<class_TextureLayeredRD>` **<** :ref:`TextureLayered<class_TextureLayered>` **<** :ref:`Texture<class_Texture>` **<** :ref:`Resource<class_Resource>` **<** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

Array de texturas para 2D que está vinculado a una textura creada en el :ref:`RenderingDevice<class_RenderingDevice>`.

.. rst-class:: classref-introduction-group

Descripción
----------------------

Esta clase de array de texturas te permite utilizar una textura de array 2D creada directamente en :ref:`RenderingDevice<class_RenderingDevice>` como textura para materiales, mallas, etc.
