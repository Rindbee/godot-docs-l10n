TextServerAdvanced
==================

**Hereda:** :ref:`TextServerExtension<class_TextServerExtension>` **<** :ref:`TextServer<class_TextServer>` **<** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

Un servidor de texto avanzado con soporte para BiDi, diseño de texto complejo y características OpenType contextuales. Utilizado en Godot por defecto.

.. rst-class:: classref-introduction-group

Descripción
----------------------

Una implementación de :ref:`TextServer<class_TextServer>` que utiliza HarfBuzz, ICU y SIL Graphite para soportar BiDi, diseños de texto complejos y características contextuales de OpenType. Esta es la interfaz :ref:`TextServer<class_TextServer>` primaria por defecto de Godot.
