PhysicsServer2DExtension
========================

**Hereda:** :ref:`PhysicsServer2D<class_PhysicsServer2D>` **<** :ref:`Object<class_Object>`

Proporciona métodos virtuales que pueden ser sobreescritos para crear implementaciones personalizadas de :ref:`PhysicsServer2D<class_PhysicsServer2D>`.

.. rst-class:: classref-introduction-group

Descripción
----------------------

Esta clase extiende :ref:`PhysicsServer2D<class_PhysicsServer2D>` proporcionando métodos virtuales adicionales que pueden ser sobreescritos. Cuando estos métodos son sobreescritos, serán llamados en lugar de los métodos internos del servidor de físicas.

Destinado para su uso con GDExtension para crear implementaciones personalizadas de :ref:`PhysicsServer2D<class_PhysicsServer2D>`.
