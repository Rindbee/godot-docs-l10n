VisualShaderNodeIf
==================

**Hereda:** :ref:`VisualShaderNode<class_VisualShaderNode>` **<** :ref:`Resource<class_Resource>` **<** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

Emite un vector 3D basado en el resultado de una comparación de punto flotante dentro del gráfico de shader visual.

.. rst-class:: classref-introduction-group

Descripción
----------------------

Este nodo de shader visual tiene seis puertos de entrada:

- Los puertos **1** y **2** proporcionan los dos números de punto flotante ``a`` y ``b`` que se compararán.

- El puerto **3** es la tolerancia, que permite que números de punto flotante similares se consideren iguales.

- Los puertos **4**, **5** y **6** son las posibles salidas, que se devuelven si ``a == b``, ``a > b`` o ``a < b`` respectivamente.
