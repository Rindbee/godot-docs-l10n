Theme
=====

**Hereda:** :ref:`Resource<class_Resource>` **<** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

Un recurso utilizado para dar estilo/personalizar los :ref:`Control<class_Control>`\ es y :ref:`Window<class_Window>`\ s.

.. rst-class:: classref-introduction-group

Descripci√≥n
----------------------

Un recurso utilizado para estilizar/skinning en nodos :ref:`Control<class_Control>` y :ref:`Window<class_Window>`. Si bien los controles individuales se pueden estilizar utilizando sus modificaciones de tema locales (ver :ref:`Control.add_theme_color_override()<class_Control_method_add_theme_color_override>`), los recursos de tema le permiten almacenar y aplicar la misma configuraci√≥n en todos los controles que comparten el mismo tipo (por ejemplo, estilizar todos los :ref:`Button<class_Button>`\ s de la misma manera). Se puede utilizar un recurso de tema para todo el proyecto, pero tambi√©n puede establecer un recurso de tema separado para una rama de nodos de control. Un recurso de tema asignado a un control se aplica al propio control, as√≠ como a todos sus hijos directos e indirectos (siempre y cuando una cadena de controles no se interrumpa).

Utiliza :ref:`ProjectSettings.gui/theme/custom<class_ProjectSettings_property_gui/theme/custom>` para configurar un tema de alcance de proyecto que estar√° disponible para cada control en tu proyecto.

Utiliza :ref:`Control.theme<class_Control_property_theme>` de cualquier nodo de control para configurar un tema que estar√° disponible para ese control y todos sus hijos directos e indirectos.
