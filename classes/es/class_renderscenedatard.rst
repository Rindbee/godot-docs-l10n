RenderSceneDataRD
=================

**Hereda:** :ref:`RenderSceneData<class_RenderSceneData>` **<** :ref:`Object<class_Object>`

Implementación de datos de escena de renderizado para los renderizadores basados en RenderingDevice.

.. rst-class:: classref-introduction-group

Descripción
----------------------

El objeto contiene datos de la escena relacionados con el renderizado de un solo fotograma de un viewport.

\ **Nota:** Este es un objeto interno del servidor de renderizado, no instancies esto desde un script.
