Editor de perfil de interacción predeterminado de OpenXR.

.. rst-class:: classref-introduction-group

Descripción
----------------------

Este es el editor de perfil de interacción predeterminado de OpenXR que proporciona una interfaz genérica para editar cualquier perfil de interacción para el que no se haya definido ningún editor personalizado.
