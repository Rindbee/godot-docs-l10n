ResourceImporterShaderFile
==========================

**Hereda:** :ref:`ResourceImporter<class_ResourceImporter>` **<** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

Importa shaders GLSL nativos (no shaders de Godot) como un :ref:`RDShaderFile<class_RDShaderFile>`.

.. rst-class:: classref-introduction-group

Descripción
----------------------

Esto importa shaders GLSL nativos como recursos :ref:`RDShaderFile<class_RDShaderFile>`, para usar con operaciones de bajo nivel :ref:`RenderingDevice<class_RenderingDevice>`. Este importador *no* maneja archivos ``.gdshader``.
