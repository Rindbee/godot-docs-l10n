OpenXRExtensionWrapperExtension
===============================

**Obsoleto:** Use :ref:`OpenXRExtensionWrapper<class_OpenXRExtensionWrapper>` instead.

**Hereda:** :ref:`OpenXRExtensionWrapper<class_OpenXRExtensionWrapper>` **<** :ref:`Object<class_Object>`

Permite implementar extensiones de OpenXR con GDExtension.

.. rst-class:: classref-introduction-group

Descripción
----------------------

**OpenXRExtensionWrapperExtension** permite implementar extensiones de OpenXR con GDExtension. La extensión debe registrarse con :ref:`OpenXRExtensionWrapper.register_extension_wrapper()<class_OpenXRExtensionWrapper_method_register_extension_wrapper>`.
