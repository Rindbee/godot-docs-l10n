IntervalTweener
===============

**Hereda:** :ref:`Tweener<class_Tweener>` **<** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

Crea un intervalo de inactividad en una animación de :ref:`Tween<class_Tween>`.

.. rst-class:: classref-introduction-group

Descripción
----------------------

**IntervalTweener** se usa para crear retrasos en una secuencia de interpolación. Véase :ref:`Tween.tween_interval()<class_Tween_method_tween_interval>` para obtener más información sobre su uso.

\ **Nota:** :ref:`Tween.tween_interval()<class_Tween_method_tween_interval>` es la única forma correcta de crear **IntervalTweener**. Cualquier **IntervalTweener** creado manualmente no funcionará correctamente.
