Un nodo para mover mallas manuales a partir de datos de :ref:`XRHandTracker<class_XRHandTracker>`.

.. rst-class:: classref-introduction-group

Descripci√≥n
----------------------

Este nodo utiliza datos de seguimiento de manos de un :ref:`XRHandTracker<class_XRHandTracker>` para posar el esqueleto de una malla de mano.

El posicionamiento de las manos se realiza creando un :ref:`XRNode3D<class_XRNode3D>` ancestro de la malla de la mano impulsado por el mismo :ref:`XRHandTracker<class_XRHandTracker>`.

Los datos de posici√≥n del seguimiento de la mano se escalan mediante :ref:`Skeleton3D.motion_scale<class_Skeleton3D_property_motion_scale>` cuando se aplican al esqueleto, lo que se puede utilizar para ajustar la mano rastreada para que coincida con la escala del modelo de mano.
